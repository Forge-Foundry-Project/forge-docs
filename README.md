# Forge Docs

Central documentation repository for the Forge Foundry ecosystem.

Forge Docs serves as the primary source of documentation, architecture references, roadmaps, development guides, and knowledge base content for all Forge Foundry projects.

---

## Purpose

The goal of Forge Docs is to provide a single location for:

* Project documentation
* Architecture documentation
* Development guides
* Roadmaps
* Design decisions
* Contributor information
* Knowledge base content
* Community resources

As the Forge Foundry ecosystem grows, this repository will become the central source of truth for all project documentation.

---

## Forge Foundry

Forge Foundry is an open-source software ecosystem focused on:

* Linux
* Rust
* Artificial Intelligence
* Developer Productivity
* Open Source Software
* Technical Education

Mission:

> Building software, sharing knowledge, and continuously learning.

---

## Projects

### ForgeOS

Immutable Linux operating system built on Fedora Atomic technologies.

### OpenForge

Open-source addon and package management platform built with Rust.

### Rust-GPT

AI-assisted developer tooling and workspace intelligence platform.

### Forge Welcome

First-run experience and onboarding platform for ForgeOS.

### Forge Assistant

AI-powered assistant focused on Linux, development workflows, and the Forge ecosystem.

### Forge IDE

Developer-focused integrated development environment.

### Forge Security

Security tooling and platform services for the Forge ecosystem.

### Forge Command Center

Centralized management and orchestration platform.
